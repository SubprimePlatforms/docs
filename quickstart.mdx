Get your first identity verification session up and running in three steps.

## Prerequisites

- A deepidv account — [sign up at app.deepidv.com](https://app.deepidv.com)
- Sufficient token balance in your organization

## Step 1: Get your API key

1. Log in to the [Admin Console](https://app.deepidv.com)
2. Navigate to **Settings** → **API Keys**
3. Click **Generate API Key**
4. Copy your key — you'll need it for every API request

<Warning>
  Keep your API key secret. Do not expose it in client-side code or commit it to version control.
</Warning>

## Step 2: Create a verification session

Make a `POST` request to create your first IDV session. This sends a verification invite to your end user via email and SMS.

<CodeGroup>
```bash cURL
curl -X POST https://api.deepidv.com/v1/sessions \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "firstName": "John",
    "lastName": "Doe",
    "email": "john.doe@example.com",
    "phone": "+15192223333"
  }'
```

```javascript Node.js
const response = await fetch("https://api.deepidv.com/v1/sessions", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY",
  },
  body: JSON.stringify({
    firstName: "John",
    lastName: "Doe",
    email: "john.doe@example.com",
    phone: "+15192223333",
  }),
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    "https://api.deepidv.com/v1/sessions",
    headers={
        "Content-Type": "application/json",
        "x-api-key": "YOUR_API_KEY",
    },
    json={
        "firstName": "John",
        "lastName": "Doe",
        "email": "john.doe@example.com",
        "phone": "+15192223333",
    },
)

print(response.json())
```
</CodeGroup>

### Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `firstName` | string | Yes | End user's first name |
| `lastName` | string | Yes | End user's last name |
| `email` | string | Yes | End user's email address |
| `phone` | string | Yes | End user's phone in E.164 format (e.g. `+15192223333`) |
| `externalId` | string | No | Your internal reference ID for this session |
| `sendEmailInvite` | boolean | No | Send an email invite (defaults to `true`) |
| `sendPhoneInvite` | boolean | No | Send an SMS invite (defaults to `true`) |
| `workflowId` | string | No | Workflow to use for this session |

## Step 3: Check the response

A successful request returns the session ID and a verification URL:

```json
{
  "idv_id": "abc123-def456-ghi789",
  "session_url": "https://app.deepidv.com",
  "links": []
}
```

| Field | Description |
|-------|-------------|
| `idv_id` | Unique identifier for the verification session |
| `session_url` | Base URL where the end user completes their verification |
| `externalId` | Your external ID (returned only if provided in the request) |
| `links` | Associated verification links |

Your end user will receive an email and/or SMS with a link to complete their identity verification.
