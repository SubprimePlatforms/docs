deepidv is a full-stack, modular identity verification platform that lets you verify identities, screen against global watchlists, and aggregate financial data — all through a single API or admin console.

Build custom verification workflows by combining only the modules you need: document verification, face liveness detection, age estimation, PEP & sanctions screening, bank statement retrieval, and more.

## How it works

<Steps>
  <Step title="Create a workflow">
    Select the verification modules you need — ID verification, liveness, sanctions screening, financial data — and save it as a reusable workflow.
  </Step>
  <Step title="Send a verification request">
    Trigger a session via API or generate a verification link to send to your end user. Each session runs the checks defined in your workflow.
  </Step>
  <Step title="Review results">
    Sessions are automatically verified or flagged for manual review. View results in the admin console, receive webhooks, or query the API.
  </Step>
</Steps>
