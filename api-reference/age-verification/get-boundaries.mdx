```
GET /v1/age-verification/parent-connect/boundaries/{attestation_id}
```

Returns the current parental boundary settings for a child's attestation. Use this to enforce the restrictions the parent configured during the Parent Connect consent flow.

## Request

### Headers

| Header      | Required | Description  |
| ----------- | -------- | ------------ |
| `x-api-key` | Yes      | Your API key |

### Path parameters

| Parameter        | Type   | Required | Description                                                |
| ---------------- | ------ | -------- | ---------------------------------------------------------- |
| `attestation_id` | string | Yes      | The child's attestation ID (format: `attest_<ULID>`)       |

### Example request

<CodeGroup>
```bash cURL
curl https://api.deepidv.com/v1/age-verification/parent-connect/boundaries/attest_01HX7Q3K2M4N5P6R7S8T9V0W \
  -H "x-api-key: YOUR_API_KEY"
```

## Response

### 200 --- Success

| Field                  | Type    | Description                                       |
| ---------------------- | ------- | ------------------------------------------------- |
| `child_attestation_id` | string  | The attestation ID queried                        |
| `parent_connect_id`    | string  | Parent Connect record ID                          |
| `parent_session_id`    | string  | Parent's verification session ID                  |
| `age_band`             | string  | Child's age band (e.g. `"13-17"`)                 |
| `boundaries`           | object  | Current boundary settings keyed by boundary `key` |
| `eula_accepted`        | boolean | Whether the parent accepted the EULA              |
| `status`               | string  | `CONSENT_GIVEN`                                   |
| `consent_expires_at`   | string  | ISO 8601 expiration timestamp (180 days)          |

### Error responses

| Status | Description                                                   |
| ------ | ------------------------------------------------------------- |
| `401`  | Missing or invalid `x-api-key`                                |
| `404`  | Attestation not found, or not linked to a Parent Connect flow |

## Update Boundaries

```
PUT /v1/age-verification/parent-connect/boundaries/{attestation_id}
```

Updates the parental boundary settings without requiring the parent to re-verify. The updated boundaries are stored on both the session and the Parent Connect record.

A `parent_connect.boundaries_updated` webhook is fired to your registered endpoint.

### Body parameters

| Parameter    | Type   | Required | Description                              |
| ------------ | ------ | -------- | ---------------------------------------- |
| `boundaries` | object | Yes      | Updated boundary settings as key-value pairs |

### Example request

<CodeGroup>
```bash cURL
curl -X PUT https://api.deepidv.com/v1/age-verification/parent-connect/boundaries/attest_01HX7Q3K2M4N5P6R7S8T9V0W \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "boundaries": {
      "in_game_purchases": true,
      "daily_time_limit": 180,
      "mature_content": false
    }
  }'
```

```javascript Node.js
const response = await fetch(
  "https://api.deepidv.com/v1/age-verification/parent-connect/boundaries/attest_01HX7Q3K2M4N5P6R7S8T9V0W",
  {
    method: "PUT",
    headers: {
      "Content-Type": "application/json",
      "x-api-key": "YOUR_API_KEY",
    },
    body: JSON.stringify({
      boundaries: {
        in_game_purchases: true,
        daily_time_limit: 180,
        mature_content: false,
      },
    }),
  }
);
```

```python Python
import requests

response = requests.put(
    "https://api.deepidv.com/v1/age-verification/parent-connect/boundaries/attest_01HX7Q3K2M4N5P6R7S8T9V0W",
    headers={"x-api-key": "YOUR_API_KEY"},
    json={
        "boundaries": {
            "in_game_purchases": True,
            "daily_time_limit": 180,
            "mature_content": False,
        }
    },
)
```
</CodeGroup>
